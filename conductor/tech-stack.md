# Technology Stack

## Core Technologies
- **Language:** TypeScript
- **Runtime:** Bun
- **Frontend Framework:** Vanilla Web Components (Custom Elements)
- **Styling:** Vanilla CSS (Modern CSS features)

## Tooling & Infrastructure
- **Bundler:** Vite
- **Linting & Formatting:** Biome
- **Type Checking:** TypeScript (TSC)

## PDF Engine
- **Processing:** pdf-lib
- **Rendering:** pdf.js

## Testing Frameworks
- **Unit & Integration:** Vitest
- **End-to-End:** Playwright
- **Coverage:** Vitest (v8/Istanbul)

## Dependencies
- **Icons:** Lucide
- **Environment:** Node.js types (for tooling)
